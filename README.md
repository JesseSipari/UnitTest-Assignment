
# Unit Test Assignment

This project contains a JavaScript module `mylib` with basic arithmetic operations and unit tests using Mocha and Chai.

## Usage

### Installation

Install the dependencies defined in package.json:

```bash
npm install

```

### Running Tests

Execute the unit tests with Mocha:

```bash
npm test

```

### Running Main Program

Run the main demonstration script:

```bash
node main.js

```
